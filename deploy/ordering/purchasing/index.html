Pasang Dashboard di HP (Android & iOS)

Ubah file HTML tunggal ini menjadi aplikasi ponsel yang bisa di-install tanpa App Store. Cukup klik tombol di bawah untuk mengunduh berkas lengkapnya, lalu tempatkan berkas tersebut pada hosting gratisan Anda!

Cara Pasang Menjadi Aplikasi HP dalam 2 Menit:

1. Ekstrak Berkas ZIP

Klik tombol hijau di atas untuk mendapatkan berkas ZIP. Ekstrak berkas tersebut, maka Anda akan mendapatkan 3 buah berkas: index.html, manifest.json, dan sw.js.

2. Unggah ke Hosting Gratis

Buka situs Netlify.com atau Vercel.com. Buat akun gratis, lalu drag and drop folder hasil ekstrak tadi langsung ke kolom unggahan Netlify Drop.

3. Buka Link & Pasang!

Buka alamat tautan website aman (https://) yang diberikan Netlify di Google Chrome (Android) atau Safari (iOS) di ponsel Anda. Klik tombol "Pasang di HP" yang muncul di navigasi atas!

Panduan Sinkronisasi Sistem Terpusat Google Sheets

Ikuti langkah-langkah di bawah ini agar input dari Google Form kitchen bisa langsung masuk ke aplikasi lokal ini secara real-time.

01

Persiapan Google Form

Buatlah Google Form untuk tim kitchen dengan minimal kolom pertanyaan berikut secara berurutan:

  • Nama Item (Dropdown)
  • Jumlah Order (Angka)
  • Satuan (Dropdown)
  • Catatan untuk Vendor (Paragraf)
  • Tanggal Rencana Kedatangan (Tanggal)
  • PIC Order Kitchen (Jawaban Singkat)
02

Pasang Apps Script

Di Google Sheets Anda:

  1. Klik menu Ekstensi > Apps Script.
  2. Anda dapat memisahkan script ke banyak file spreadsheet atau menyatukannya dengan mengarahkan URL tujuan masing-masing di form bawah ini.
  3. Gunakan kode Apps Script v9.9+ yang mendukung API terdistribusi.
  4. Klik ikon Simpan (disket).
03

Deploy Aplikasi Web

Aktifkan API di internet:

  1. Klik tombol Deploy > New deployment.
  2. Klik ikon gigi roda, pilih Web app.
  3. Atur Who has access menjadi: Anyone.
  4. Klik Deploy dan berikan izin otorisasi.
  5. Lakukan ini untuk setiap spreadsheet jika terpisah.
